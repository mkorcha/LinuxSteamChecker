Linux Steam Checker

What in your library will run natively on Linux? Enter your Steam profile URL below and find out!

This page was created for those considering making the switch to Linux full time, who want to know what games in their library will go with them. Don't see your favourites listed? Don't worry, you still have options! Wine allows you to run many games using a Windows compatibility layer, and has a strong community willing to help. You can also create a gaming VM and run a Windows VM at near-native performance, if your hardware supports it. Finally, you could always dual boot, however it can be inconvenient.
