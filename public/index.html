Linux Steam Checker

Want to see what you have that will run natively? Enter your Steam profile URL below and find out!

This page was created for people considering switching to Linux full-time and want to see what they can take with them. Don't see all of your favourites? Never fear, you still have options! Wine allows you to run many games using a compatibility layer, and often you can find support for your favourite games to get it set up. You can also create a gaming VM and run a Windows VM only for games at near-native performance, if your hardware supports it. Finally, dual booting is also an option, though it can be inconvenient.
